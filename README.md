# AirHockey

Create a full fledged Air Hockey Mobile Game with dual player support. Two players will play with the same touch device. Make any two of the given versions of the game: 1. timed game for 1-2 minutes 2. First to 10 goals wins 3. Best of three 1 minute timed game. Bonus: Make the scoreboard like a grid. Bonus: Make effects when a player wins or losses.
